Opdracht 1

Wat zijn de eigenschappen van block boxes en inline boxes?

Block Element:

Een block element is een element dat op een nieuwe regel begint en de volledige breedte van de pagina of container gebruikt. Een block element kan een of meerdere regels in beslag nemen en heeft een line break voor en na het element.

Inline Element:

Inline elementen worden in een regel weergegeven. Ze dwingen de tekst erna niet naar een nieuwe regel. Een anchor-tag is een voorbeeld van een inline element. Je kan meerdere links op een rij zetten wat dat op een regel wordt weergegeven.
