Wat is Cascade?

Cascade betekent dat de volgorde van de CSS-regels ertoe doen; CSS wordt gelezen en uigevoerd van boven naar beneden. Dit betekent dat wanner je twee regels hebt en dezelfde stijling geeft aan een element, de laatst geschreven regel zal worden uitgevoerd.
Bijvoorbeeld: h1 { color: red; } , h1 { color: blue; } de h1 zal dan een blauwe kleur hebben.

Wat is Inheritance?

In CSS inheritance is wanneer een style door sommige child elements wordt overgenomen van de parent element.
Bijvoorbeeld, als je een element een kleur en een font-family property geeft zal alle inhoud van het element worden gestyled. Tenzij je die specifieke content een andere kleur en een andere font-family hebt gegeven.
