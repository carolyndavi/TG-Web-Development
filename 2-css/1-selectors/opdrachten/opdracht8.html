Wat is Cascade?

Cascade betekent dat de volgorde van de CSS-regels ertoe doen; CSS wordt gelezen en uigevoerd van boven naar beneden. Dit betekent dat wanner je twee regels hebt en dezelfde stijling geeft aan een element, de laatst geschreven regel zal worden uitgevoerd.
Bijvoorbeeld: h1 { color: red; } , h1 { color: blue; } de h1 zal dan een blauwe kleur hebben.
