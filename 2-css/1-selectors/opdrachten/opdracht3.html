Type Selector:

Een type selector koppelt elementen aan de corresponderende naam van het element. Type selectors worden over het algemeen gebruikt om grote veranderingen aan te brengen aan de stijl van een site.
Bijvoorbeeld: h1 { color : blue; }
Het voordeel is dat je alle tags met dezelfde naam in een document kunt veranderen.


Descendant Selector:

De descendant selector is een manier om elementen te selecteren die zich ergens onder andere elementen bevinden. Deze selector is eigenlijk meerdere selectors gecombineerd, maar gescheiden door een spatie.
Bijvoorbeeld: ul li { color : red; }
Het voordeel van een descendant selector is dat je de "children" van een tag kan veranderen.


Class Selector:

Een class selector gebruik je door een punt voor de selector naam te plaatsen. Het selecteert alle elementen met dat class-attribuur.
Bijvoorbeeld: .class { color : green; }
Het voordeel van een class selector is dat je ze kan hergebruiken op verschillende plaatsen en zelfs verschillende pagina's.
