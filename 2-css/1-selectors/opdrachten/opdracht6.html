Child Selectors:

Een child selector in CSS is het "groter dan" symbool, het ziet er als volgt uit: ol > li.
Dit betekend "selecteer elementen die alleen directe afstammelingen zijn". In this geval, "selecteer lijstitems die direct afstammen van een geordende lijst".
Bijvoorbeeld: ol > li { color : red; }
Het voordeel van een child selector is dat het modelijk is elementen te selecteren die een parent > child relatie hebben.

Adjacent Selectors:

Een adjecent selector in CSS is op zichzelf geen selector, maar een manier om twee selectors te combineren. Een adjecent selector wordt gebruikt om een element te selecteren dat direct na een ander specifiek element staat. Sibling elementen moeten hetzelfde parent element hebben en "adjacent" betekend "direct volgend".
Bijvoorbeeld: div + p { color : blue; }
Het voordeel van een adjectent selector is dat je een element kunt selecteren die precies naast elkaar liggen.

General Selectors:

De general selector wordt gebruikt met deze symbool: ~. Deze selector scheidt twee selectors en komt overeen met alle iteraties van het tweede element, die het eerste element volgen (hoewel het niet onmiddelijk is), die de children van dezelfde parent zijn.
Bijvoorbeeld: img ~ p { color : green; }
Het voordeel van een general selector is dat je elementen kunt matchen die elkaar volgen.
