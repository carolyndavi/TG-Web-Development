Opdracht 1


Inline CSS:

Dit is CSS-code dat je in een HTML-element tag schrijft. Inline CSS zal daarom ook alleen het element stijlen waar de CSS is in geschreven. Als je bijvoorbeeld meerdere “p” elementen hebt, zal dus dan alleen de “p” element waarin je de CSS hebt geschreven worden aangepast. Alle andere “p” elementen zullen niet worden gestijld. Tenzij je die specifieke elementen inline CSS-styling geeft.


Internal CSS:

Interne CSS wordt gebruikt om een stijl te definiëren voor een enkele HTML-pagina. Deze CSS-code schrijf je in het hoofdgedeelte van het HTML-document. Je gebruikt dan de style tag, en tussen de open en gesloten style tag schrijf je je CSS-regels. Als je met Internal CSS werkt en dan nu bijvoorbeeld de “p” element styling geeft, zullen alle “p” elementen die je op je webpagina hebt gestijld worden.


External CSS:

Een externe CSS-document is een aparte CSS-bestand dat toegankelijk is door een link te maken in het hoofdgedeelte van het HTML-document. Je creëert een nieuw document dat dan eindigt op .css en in dat CSS-document schrijf je je CSS-regels. Zo kan je meerdere webpagina’s dezelfde styling geven.
