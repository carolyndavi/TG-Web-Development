Selectors & Visual Rules

Opdracht 2

CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

selector { property : value; }

Het eerste deel is de selector, dit komt aan het begin van elke CSS-regel. De selector is het HTML-element dat je wilt gaan stijlen. Dus welke HTML-element wil je veranderen? Op de plek van de selector kan je dan bijvoorbeeld <h1> typen.

Dan komen er een paar accolades, tussen de open en gesloten accolade zal je je CSS-regels schrijven.

De property is de wat. Dus wat wil je veranderen aan het HTML-element. Is het de achtergrond kleur? Is het de text? Je zou dan bijvoorbeeld: background-color kunnen typen op de plek van property.

Na de property type je dubbele punt, om aan te geven dit wil ik naar dat veranderen.

De value is dan de waarde waar je het element naar wilt veranderen. Als de property dan background-color is, zou je de waarde als blue kunnen zetten. Je wilt dan de achtergrond naar blauw veranderen van de <h1>.

De regel zal het uiterlijk van het HTML-element veranderen en het een nieuwe waarde geven. Elke regel moet eindigen met een puntkomma.
