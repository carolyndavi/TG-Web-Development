Transitions / Animations

Opdracht 1

Zoek uit wat transitions zijn en beschrijf met eigen woorden wat het betekent.

Wat is CSS transitions?

De CSS transition is de makkelijkste manier om animaties toe te voegen aan een website.
Met transitions defineer je de transition tussen twee elementen.


Welke properties zijn er?

transition-property:

Specificeert de naam of namen van de CSS properties die geanimeerd moeten worden. Alleen de properties die worden vermeld worden geanimeerd.

transition-duration:

Specificeert de duratie van hoelang een transition moet zijn. Je kan een duration toevoegen aan alle properties tijdens de transition.

transition-timing-function:

Dit zorgt ervoor dat alle animaties precies de zelfde tijd nodig hebben om klaar te zijn. Maar het verschil tussen de transition-timing-functions is dat hoe de door de transition gaan anders is. Bijvoorbeeld begint de transition langzaam en eindigt snel, of begint de transition snel en eindigt langzaam.

transition-delay:

Definieerd hoe lang een property moet wachten voordat de transition begint.
