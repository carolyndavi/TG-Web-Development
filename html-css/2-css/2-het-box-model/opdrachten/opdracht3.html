Het Box Model

Opdracht 3

Wat gebeurt er als je een width en een height toevoegt aan de span tag ( <span> ) van vorige opdracht?

Antwoord:

Er gebeurt niks. Spans zijn inline elementen. Het heeft daarom geen breedte of hoogte. Als je de breedte of de hoogte zou willen aan passen dan zou je van de span een block-level element van moeten maken.
