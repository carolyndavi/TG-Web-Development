Er zijn 14 verschillende pseudo elementen. Je hebt in de voorgaande opdracht reeds 4 kunnen benoemen. Kun je de andere 10 ook benoemen en aangeven wat ze precies doen?

De andere pseudo elementen:

::backdrop is een experimentele pseudo-element dat creÃ«ert een backdrop dat de gehele viewport inneemt en wordt direct onder een <dialog> weergegeven.

::cue is een pseudo-element die je kunt gebruiken om media captions stijlen.

::first-letter is een pseudo-element die je kunt gebruiken je de eerste letter van een block element te stijlen.

::first-line is een pseudo-element die je kunt gebruiken om de eerste regel van een block element te stijlen.

::file-selector-button is een pseudo-element die je kunt gebruiken om een button te stijlen waar je op kunt clicken om bestanden te uploaden.

::grammar-error is een experimentele pseudo-element die je kunt gebruiken om een element voor een ander element te plaatsen.

::placeholder is een pseudo-element die je kunt gebruikten om text in inputs of textareas te plaatsen.

::spelling-error is een experimentele pseudo-element die je kunt gebruiken wanneer je een fout hebt geschreven in je text kan je het markeren.

::target-text is een experimentele pseudo element die je kunt gebruikten om tekst waarnaar is gescrolled als de browser scroll-to-text onderstuint. Hiermee kunnen auteurs kiezen hoe ze dat gedeelte van de tekst willen markeren.
