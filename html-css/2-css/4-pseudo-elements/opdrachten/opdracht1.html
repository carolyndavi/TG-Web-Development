Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)? Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

Wat zijn Pseudo Elements

Pseudo elements zijn een goeie manier om extra stijl toetevoegen aan je website zonder dat je extra html hoeft te schrijven. Pseudo Elements zijn inline-elementen.

Pseudo Elementen:

::after is een pseudo-element dat er voor zorgt dat je een extra laag kan plaatsen na de geselicteerde element-content.

::before is een pseudo-element dat er voor zorgt dat je een extra laag kan plaatsen voor de geselicteerde element-content.

::selection is een pseudo-element dat er voor zorgt dat wanneer je tekst selecteerd, je de stijling kan veranderen. Bijvoorbeeld de cursor of de achtergrond kleur. Je kan zelfs een tekst schaduw implementeren.

::marker is een pseudo-element dat de stijling van lijst markers veranderd.
