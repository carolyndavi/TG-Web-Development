Opdracht 1

We hebben nu zowel mixins als extends gezien die beiden styling kunnen overnemen. Lees dit artikel en geef een uitleg wanneer we inheritance gebruiken en wanneer we mixins gebruiken.

Antwoord:

Je kan met extend ook code hergebruiken. Het ligt eraan wat je probeerd te doen. Maar meestal als je geen dynamische stijl gebruikt in je SCSS bestand dan gebruik je inheritance. Wanner je wel dynamische stijlen gaat gebruiken, dan gebruik je mixins.
